References

The reference for the logo and the navigation bar was taken Ximena Vengoechea's website. In both the websites, the logo has the name of the author and is placed at the center. The referenced website has navigation bar right below the logo. The same concept has been used, but in the project website the navigation bar has a different background and the logo has only the author's name.



The colours of the website has been referred from Cristina Vanessa's website. The hex values of the colours used in the website are #f3ede9 for the background, #958271 for the links, #7185a8 for the link when it is being hovered, #e395af for the link wehn it is being clicked, #d1c9c9 for the background of the navigation bar, #d5d5d5 and #f7d5ef for the background of certain parts of the website.



The insipiration for box shadow was taken from Alfred App's website. The shadows have been used in different parts of the website. This element makes certain part of the webpages pop out. For this feature, a class has been created in the CSS. The class contains box-shadow element with a grey color and dimensions of 0px 10px 30px 0px.



The reference for the footer section was taken from Bob Malgirini's website. Both the websites have a background occupying the whole footer section with icons of social media which directs to the author's profile. The reference website has white icons for instagram, twitter, linkedin and dribble whereas the project website has black icons for facebook and instagram.



The form for feedback was inspired from the website THE Sum. The reference website has fields to fill the name, email and the feedback. In the project website, fields to insert the first name, last name, address, phone number, email and the feedback has been added. Both the forms are in a separate box. Box shadow has been used in the project website.
